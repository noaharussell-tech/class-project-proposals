Application Overview

ClassForge lets students and instructors quickly create and share class project proposals using a simple web interface. Users can fill in product details, feature lists, and export their proposals to PDF or share via link.

Technology Overview

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js (optional)
  • Hosting: GitHub Pages or Netlify
