GameGuesser - Product Proposal

Product Name

GameGuesser - Top-Selling Games Quiz

Team Members

  • Noah Russell

Application Overview

GameGuesser is a desktop-based quiz game built with Python and Tkinter. The game challenges the user to guess which of two top selling video games has sold more copies worldwide. Each round randomly selects two games from the top 10 best selling games of all time, and the player chooses which one they believe sold more. Hints and fun facts are included for extra engagement, and a final score is displayed at the end.

Technology Overview

  • Programming Language: Python
  • GUI Framework: Tkinter
  • Development Environment: Visual Studio Code
  • No Web Hosting: Runs as a standalone application

Proposal Last Updated

December 8, 2025

Feature List (Prioritized)

  1. Fully functional Tkinter windows application
  2. Randomized pair generation each round
  3. Accurate comparison using real-world sales numbers
  4. Hints for each game to assist the player
  5. Fun facts displayed after every guess
  6. Total score summary at the end of the game
  7. Replay system that resets the entire game

Top Game Dataset Used

The quiz includes the following top-selling games (units sold in millions):

  • Tetris - 520M
  • Minecraft - 300M
  • Grand Theft Auto V - 220M
  • Wii Sports - 82.9M
  • PUBG - 75M
  • Mario Kart 8 / Deluxe - 70M
  • Red Dead Redemption 2 - 67M
  • Skyrim - 60M
  • Super Mario Bros. - 58M
  • The Witcher 3 - 50M

Basic Technical Features

  • Python event-driven GUI using Tkinter
  • Button interactions for user choices
  • Dynamic text updates (feedback, hints, facts)
  • Score tracking and display

Advanced Technical Features

  • Random procedural generation of quiz rounds
  • Game data stored as a structured Python dictionary
  • Dynamic color changes based on correct/incorrect answers
  • End-of-game screen replacement via widget destruction

Future Feature List

  • Ability to import updated game sales data automatically
  • Multiple quiz modes (e.g., yearly charts, genre-based quizzes)
  • User-selectable custom quiz lists
  • Audio feedback and animations
  • Leaderboard or high-score system
  • Packaging into a downloadable .exe installer

Running Instructions: Run gameguesser.py in Python 3.10+ to launch the application.
